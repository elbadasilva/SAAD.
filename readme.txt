Instruções de execução:

No Prompt de Comando do Windows ou no terminal do Linux, digite "python main.py nome_do_arquivo.json".
Cada arquivo JSON simula diferentes perfis de compradores dos módulos:

a) Alunos que não compraram nenhum módulo: teste1_não_comprou.json;
b) Alunos que compraram à vista.json: teste2_a_vista.json;
c) Alunos que compraram parcelados: teste3_parcelado.json;
d) Alunos que não comparam todos os itens: teste4_alguns_itens.json.
